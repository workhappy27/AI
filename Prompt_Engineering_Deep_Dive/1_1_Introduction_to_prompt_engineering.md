#### What is prompt engineering?  

Prompt engineering: Descipline for engineering prompts.  
The goal is to desing good prompts.  

Why Prompt engineering?  
 - LLMs output are uncertain.
 - A systematic approach for searching solutions for problems.  

#### What is a prompt?  
- Prompt is a interface to convey user's intention to the LLM, thorugh texts, voice etc.
- It can be question, an instruction, request.

#### Prompt Basics:  
Components of Prompt: Instruction, context,  input data, output indicator.  

#### Prompt Engineering Template:  
- Define your task clearly
- Define the Evaluation metric
- Generate candidate prompts
- Experiment
- Learning when to stop experimenting

#### Prompt Engineering Best Practices:  
Prompt Engineering Guide:  
- OpenAIs guide for building good prompts.  
  - Strategy-1: Write clear instructions.  
  - Strategy-1: Provide reference text.  
  - Strategy-3: Break tasks into subtasks.
  - Strategy-4: Give the model time to think.
  - Strategy-5: Use external Tools.  
  - Strategy-6: Test changes systematically.
